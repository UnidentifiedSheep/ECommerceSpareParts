# Roadmap

## Pricing

- [ ] Рассчитывать логистику по календарям поставщика и склада.
- [ ] Подключить Armtek и TruckMotors.
- [ ] Получать рекомендацию Analytics и учитывать её вместе с бизнес-правилами при расчёте итоговой цены.

## Main

- [ ] Обрабатывать ответы поставщиков и обогащать ими каталог.
- [ ] Предлагать создание и переименование продуктов, производителей и алиасов.
- [ ] Предлагать аналоги и автоматически создавать их при достаточной уверенности.
- [ ] Добавить редактирование и подтверждение supplier-названий производителей.
- [ ] Публиковать события при подтверждении или изменении supplier-референсов.

## Analytics

- [ ] Хранить и анализировать историю цен поставщиков.
- [ ] Прогнозировать цену, дату её изменения, уверенность и срок действия прогноза.
- [ ] Формировать рекомендации для Pricing с учётом цен, продаж и маржинальности.
- [ ] Добавить фоновые задачи анализа и пересчёта цен.
- [ ] Добавить ABC-, XYZ- и совместный ABC/XYZ-анализ.

## Search

- [ ] Добавить suggestions по названию, артикулу, производителю и алиасам.
- [ ] Исправлять опечатки и поддержать транслитерацию.
- [ ] Искать по подтверждённым supplier-названиям производителей.
- [ ] Обновлять индекс после подтверждения или изменения названия.
