# Roadmap

## Pricing

- [x] Добавить конфигурируемый pipeline ценовых правил с локальными и динамическими PriceApplier.
- [x] Добавить CRUD, получение конфигурации, версионирование и инвалидацию кэша PriceApplier.
- [x] Запускать пересчёт цен при изменении markup, pricing settings, валютных курсов и PriceApplier.
- [ ] Рассчитывать логистику по календарям поставщика и склада.
- [ ] Подключить Armtek и TruckMotors.
- [ ] Получать рекомендацию Analytics и учитывать её вместе с бизнес-правилами при расчёте итоговой цены.

## Main

- [x] Добавить организации, участников, роли и управление организациями.
- [x] Создавать отсутствующие системные организации для системных пользователей через seed.
- [x] Перенести балансы, финансовые профили и транзакции с пользователей на организации.
- [x] Привязать продажи и закупки к организации и пользователю с проверкой membership.
- [x] Перенести продуктовые резервации на организации.
- [ ] Обрабатывать ответы поставщиков и обогащать ими каталог.
- [ ] Предлагать создание и переименование продуктов, производителей и алиасов.
- [ ] Предлагать аналоги и автоматически создавать их при достаточной уверенности.
- [ ] Добавить редактирование и подтверждение supplier-названий производителей.
- [ ] Публиковать события при подтверждении или изменении supplier-референсов.
